# PRD — Palm d'Or Dakhla

## Problème

Les voyageurs à Dakhla cherchent des hébergements et services sur Google. Palm d'Or n'a pas de présence digitale structurée pour capter ces visiteurs et les convertir en réservations.

## Solution

Un site vitrine premium, multi-services, optimisé conversion, dont le seul objectif est de pousser le visiteur vers une réservation WhatsApp.

## Utilisateurs cibles

- Couples marocains en weekend à Dakhla
- Familles en vacances (4–5 personnes)
- Kitesurfers et voyageurs sportifs
- Voyageurs d'affaires de passage

## Objectif produit

Générer des leads qualifiés (nom + téléphone + service) capturés en base de données et redirigés vers WhatsApp avec un message pré-rempli.

## Métriques de succès

- Clics WhatsApp (événement `Contact` Meta Pixel + GA4)
- Soumissions formulaire (événement `Lead`)
- Taux de conversion visite → WhatsApp
- Nombre de leads en base Supabase

## Périmètre V1 (livré)

- Homepage optimisée conversion
- 4 pages services (hébergements, restaurant, café, location voiture)
- Formulaire connecté Supabase
- Redirection WhatsApp avec message dynamique par service
- Structure tracking Meta Pixel + GA4
- Design mobile-first premium

## Hors périmètre V1

- Système de réservation en ligne (paiement)
- Calendrier de disponibilité temps réel
- Dashboard admin
- Version anglaise du site
- Automatisation CRM
