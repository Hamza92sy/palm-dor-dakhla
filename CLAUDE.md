# PROJECT: Palm d’Or Dakhla

## Objective
Build a premium bilingual (FR/EN) website for Palm d’Or Dakhla (restaurant, café, accommodations).

The goal is to convert Google visitors into WhatsApp reservations.

---

## Phase 1 Scope
- Homepage
- Restaurant page
- Café page
- Hébergements page
- Galerie page
- Contact page
- WhatsApp CTA
- Contact form
- Google Maps
- Mobile-first design
- Basic SEO

---

## Excluded (Phase 1)
- Stripe
- Booking system
- Supabase
- Admin dashboard
- Availability system

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- TailwindCSS
- Vercel

---

## Routing (IMPORTANT)
Use bilingual routing:

/fr
/en

---

## Design Direction
- Premium
- Minimal
- Elegant
- Hospitality style
- Colors: sand / blue / gold
- Large images
- Clean spacing

---

## Core UX Goal
User flow must be:

Google → Website → WhatsApp → Reservation

---

## Rules
- Build reusable components
- Keep code clean and scalable
- Do NOT over-engineer
- Focus on conversion (WhatsApp clicks)
- Mobile-first always

---

## Workflow
1. Build layout (navbar + footer)
2. Build homepage (priority)
3. Build other pages
4. Optimize UI
5. Prepare for deployment

---

## Important
Before coding:
- Always understand the task
- Keep design premium
- Avoid unnecessary complexity
